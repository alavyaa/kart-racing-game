# 🏎️ Kart Racing Game

A fast-paced 3D kart racing game built with Unity where players race against opponents, drift through corners, and compete for the fastest lap times.

## 🎮 Features

- 🚗 Smooth kart driving mechanics
- 🏁 Multiple racing tracks
- ⚡ Boost and acceleration system
- 🎯 Checkpoint and lap tracking
- 🤖 AI-controlled opponents
- 🎥 Dynamic camera system
- 🎨 3D environments and visual effects


## 🛠️ Built With

- **Unity**
- **C#**
- Unity Physics System
- Unity Input System

## 🚀 Getting Started

### Prerequisites

Before running the project, make sure you have:

- Unity Hub
- Unity Editor (recommended version used for development)
- Git


## 🎯 Controls

| Action | Key |
|----------|-----|
| Accelerate | W / Up Arrow |
| Brake / Reverse | S / Down Arrow |
| Steer Left | A / Left Arrow |
| Steer Right | D / Right Arrow |
| Drift | Shift |
| Boost | Space |
